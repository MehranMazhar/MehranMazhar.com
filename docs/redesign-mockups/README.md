## Critic notes (summary)

- **Bold Editorial** — most distinctive (9/10). Magazine identity: serif display, outline numerals,
  hairline grid, terracotta accent on warm paper. Watch: some body copy set in muted grey (contrast).
- **Clean Light** — most readable (9/10), enterprise-credible. Deep-teal accent on off-white.
  Watch: composition is a common dev-portfolio layout (lower distinctiveness).
- **Refined Dark** — safest evolution of the current look. Teal-on-near-black, numbered sections.
  Watch: reads as the canonical dark-portfolio template (distinctiveness 6).
- **Terminal / Engineer** — cohesive IDE aesthetic, file-path section headers. Watch: lots of small
  low-contrast monospace; most well-worn "engineer portfolio" trope.

Recommendation: **Bold Editorial** for memorability, or **Clean Light** if you want maximum
recruiter-safety. Both scored well and are the most different from the current site.
